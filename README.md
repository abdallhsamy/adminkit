## installation
```bash
git clone git@github.com:abdallhsamy/adminkit.git
```
```bash
cd adminkit
```
```bash
composer install --ignore-platform-reqs 
```
```bash
npm install
```
```bash
npm run dev
```
```bash
cp .env.example .env
```
